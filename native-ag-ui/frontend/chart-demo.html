🚀 AG-UI 动态UI组件演示

这个页面演示了如何在AG-UI中生成各种动态UI组件，包括图表、进度条、任务列表等。 点击下面的按钮来查看不同类型的组件效果。

📈 数据可视化图表

动态生成数据图表，支持自动识别数据并创建可视化组件。

📊 进度追踪器

自动识别进度相关内容并生成进度条组件。

✅ 任务列表

自动识别任务相关内容并生成交互式任务列表。

💻 代码块

自动识别代码内容并生成带语法高亮的代码块组件。

🎛️ 状态仪表板

生成系统状态监控仪表板，显示关键指标。

📖 使用说明

如何在聊天中触发动态UI组件：

  • 图表： 发送包含数字数据的消息，如 "销售额：100 200 150 300 250"
  • 进度条： 发送包含百分比的消息，如 "项目完成进度 75%"
  • 任务列表： 发送包含任务关键词的消息，如 "今日任务：1. 完成报告 2. 开会讨论"
  • 代码块： 发送用 ``` 包围的代码内容
  • 状态监控： 发送包含状态、监控等关键词的消息

提示： 动态UI组件会自动识别消息内容并生成相应的可视化组件。
